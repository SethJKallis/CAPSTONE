Route Description

Method Path HTTP Method & Description
Fetch/Users /users GET: gets all users
Fetch/User /users/:id GET: gets a single user
Create/User /register POST: adds new user
Login/User /login POST: retrieve a valid users
Update/User users/:id POST: update existing user
Delete/User /users/:id DELETE: remove a user
